# lookandsay

My solutions to generate the ["Look-and-say" sequence](https://en.wikipedia.org/wiki/Look-and-say_sequence), e.g.

    1
    11
    21
    1211
    111221
    312211
    13112221
    1113213211

## XSLT Solution

Run using any XML file as the input, e.g.

    xsltproc lookandsay.xslt lookandsay.xslt
